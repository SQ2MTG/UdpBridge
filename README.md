# UdpBridge
HRDLOG.net UDP bridge for QARTest, N1MM logger, Win-test, DM780

INSTALLATION INSTRUCTIONS
=========================

    Unzip these files into a folder on your hard drive (e.g. C:\Program Files\UdpBridge)
    Run UdpBridge.exe
    Set Properties
